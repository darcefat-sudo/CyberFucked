## Особенности

### 🎮 Инициатива
- У каждого персонажа есть **ОД (Очки Действий)** — от 2 до 5 в зависимости от класса.
- **Раунд не заканчивается**, пока у кого-то остались ОД.
- Ходы идут по кругу: сначала **дальники**, затем **саппорты**, затем **ближники и танки**, затем снова **саппорты**.
- Быстрые персонажи (Охотник, Наркодрон) реально делают **больше дел** за раунд, чем медленные (Чтец).
- Когда у **всех** ОД = 0 — раунд заканчивается, ОД восстанавливаются, начинается новый круг.

### 💀 СР (Связь с Реальностью)
- Не мана, не ХП. **Рассудок.** Тратишь на способности — сходишь с ума.
- Восстанавливается едой, наркотой, отдыхом.
- Упал до 0 — персонаж сходит с ума.

### ⚔️ Активная защита
- Защита — это **действие** (тратит ОД).
- **Не защищаешься = получаешь урон автоматически.**
- Есть рискованное **Отражение** (2d6 без Сопротивления, не тратит ОД).

### 🎭 Боги-покровители
- Sleepy God (сон, ступор, метки).
- D-Day God (разрушение, судьба, проклятия).
- Электробог и Аудиобог (в разработке).
**Все боги находятся в процессе разработки.**
### 💾 Дискеты и импланты
- Прокачка через риск. Нашёл Зелёную — безопасно. Красную — можешь умереть или стать богом.

## Как играть (кратко)

1. **Создай персонажа** (выбери класс, раскидай характеристики).
2. **Бросай d20 + модификатор** для проверок.
3. **В бою:**
   - У тебя есть ОД (2-5).
   - В свой ход тратишь 1-2 ОД на действия (Атака, Защита, Рывок, Подготовка и т.д.).
   - Ходишь по кругу, пока у всех не кончатся ОД.
   - Новый раунд — ОД восстанавливаются.
4. **Следи за СР.** Кончится — твой персонаж сойдёт с ума.
