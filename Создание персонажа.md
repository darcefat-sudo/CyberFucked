## Шаг 1. Выбери класс
Классы делятся на **2 типа**:
#### Атака
Превосходные бойцы ближнего боя и основа любой потасовки.

| Класс             | Стиль                                 |
| ----------------- | ------------------------------------- |
| **Самурай**       | Тяжёлый танк, высокий урон, много ХП  |
| **Охотник**       | Быстрый боец, скрытность, уклонение   |
| **Контрабандист** | Универсал, может стать кем угодно     |
| **Аудиофил**      | Звуковые атаки, иммунитет к оглушению |

#### Информация
Хороши во взломе, помощи и дальнем бою. Крайне полезны в интеллектуальных задачах.

|Класс|Стиль|
|---|---|
|**Освобождённый**|Бывший зомби системы, чувствует технологии|
|**Кодер**|Хакер, взлом систем, контроль врагов|
|**Чтец**|Ходячая библиотека, баффер, знания|
|**Наркодрон**|Дрон-поддержка, борется с собой, лечит|

## Шаг 3. Посчитай СР и ХП
СР (Связь с Реальностью) = ДУХ * 3 = ...
ХП (Здоровье) = ТЕЛО * 3 = ...

## Шаг 4. Посчитай пул очков для распределения по подхарактеристикам
У каждой из трёх характеристик твоего персонажа есть **подхарактеристики**. Чтобы прокачать то, что ты хочешь, получи очки по следующим формулам:
ДУХ * 1 = ...
ТЕЛО * 1 = ... если ты НЕ персонаж класса Атака, иначе: ТЕЛО * 1.5 = ...
МОЗГ * 1 = ...

## Шаг 5. Распредели очки 
- Очки прибавляются **поверх Базы**.
- **В Сопротивление вкладывать НЕЛЬЗЯ** (оно всегда = База).
**Охотник:** +1 к Восприятию для расчёта дальности обзора (классовый бонус).
**📏 Дальность обзора = Восприятие × 15 метров**

|Условие|Модификатор|Пример (Восприятие 6 = 90м)|
|---|---|---|
|Ясный день|×1|90 м|
|Сумерки / Лёгкий дождь|÷1.5|60 м|
|Темнота|÷2|45 м|
|Густой туман / Ливень|÷3|**30 м**|
|Полная тьма|0|Только в упор|
